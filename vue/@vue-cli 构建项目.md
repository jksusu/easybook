## 使用 @vue/cli 构建项目
> 安装 vue/cli 首先的安装 node8.9版本以上
* `npm install -g @vue/cli` 安装
* `vue --version`   检测安装版本

## 创建一个项目
* `vue create hello-world` 创建一个 hello-world 项目
* `npm run serve` 开启服务

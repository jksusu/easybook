## 谷歌扩展工具
* JSONView 将json数据转成数组。简介明了
* 广告终结者 如名
* Standardized Screenshot 截图神器
* Octotree  在 GitHub 左侧显示当前项目的目录结构
* Postman   开发专用，免去安装
* Adguard   广告拦截器
* GITHUBER  如果你喜欢开源，这款挺不错，打开新页面会有一个 github 开源项目排名等等。
* Dark Reader 一键切换黑夜模式，支持自定义。
* 图流    直接可以看到当前网页全部图片，避免一张一张翻
* MaoXian Web Clipper 剪辑网页，保存本地支持 html Markdown格式
* Wappalyzer  能看到当前网站所用技术栈
* Click&Clean  关闭浏览器清除缓存，保护隐私
* 划词翻译   划词翻译

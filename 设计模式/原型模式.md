## 原型模式

> php中有个魔术方法 __clone ，如果定义了 __clone 方法，则新创建的对象复制已创建的对象。这样在创建大对象时，可以直接复制。

```php
<?php
class User
{
    public $name = "cat";
}

$cat = new User();
var_dump($cat).PHP_EOL; 
// 复制一个对象
$dog = clone $cat;
var_dump($dog).PHP_EOL;

$dog->name = 'dog';
var_dump($dog).PHP_EOL;

//顺序输出
/*object(User)#1 (1) {     
  ["name"]=>             
  string(3) "cat"        
}                        
object(User)#2 (1) {     
  ["name"]=>             
  string(3) "cat"        
}                        
object(User)#2 (1) {     
  ["name"]=>             
  string(3) "dog"        
}*/
// 可以看到复制的对象明显不是同一个对象，再修改属性值后，对象还是原对象
```
### 总结
> 虽然暂时我也不知道这种模式的应用场景，不过可以在初始化频繁创建对象的时候使用原型模式，节约下系统启动的时间。
